# Linked List Processor 🔗

C++ program that stores and manipulates multiple linked lists in an array, processing a command-driven set of operations including merge, intersection, and search.

## Features
- Stores up to 100 linked lists in an array of node pointers
- Insert at head and tail, delete nodes, delete entire lists
- Find largest element in a list
- Merge two lists into a third
- Find intersection of two lists
- Sum all multiples of 5 across all lists
- Append sum of a list to its tail
- Full command-driven execution from a data file

## Tech Stack
C++ · Linked Lists · Pointers · Dynamic Memory · File I/O

## 🚀 Run
```bash
g++ DGanpat816044402.cpp -o lists
./lists
```

Ensure `asg2.txt` is in the same directory.
